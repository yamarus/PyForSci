1. Введение. Установка интерпретатора Python¶

Презентация лекция №1

1.1. О языке Python¶

Python является высокоуровневым интерпретируемым языком программирования. Подробную документацию, инструкции для новичков, установочные файлы интерпретатора и многое другое вы можете найти на pyrthon.org.

1.2. Установка интерпретатора Python¶

1.2.1. Установка Anaconda на свой компьютер¶

В рамках данного курса рекомендуется использовать дистрибутив с интерпретатором, обширным набором подключаемых модулей и средой разработки Anaconda.

  • Ссылка для скачивания Anaconda
  • Инструкция по установке Anaconda
  • Руководство пользователя Anaconda (тут есть множество подробных инструкций по использованию компонентов anaconda)

1.2.2. Установка стандартных средств для python разработки¶

Если вы по каким то причинам решили не устанавливать anaconda, а пользоваться стандартными средствами python (это требует больше усилий и времени для установки), то вам как минимум потребуется установить интерпретатор Python с оффициального сайта (после установки интерпретатора вам придется дополнительно установить математические и научные модули с помощью менеджера пакетов PyPI), и какую нибудь среду разработки (например, PyCharm, Spyder, Jupyter Notebook).
